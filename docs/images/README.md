# Images used by the viewer

## `leah-rae-mariotti-reed.jpg`

Portrait for the **Dedication** modal. Square crop recommended (the viewer
displays it in a 160px circular frame). 400×400 px or larger renders crisp on
HiDPI; JPEG at ~85% quality keeps the file under ~80 KB.

Until this file is added, the modal shows a warm "LR" monogram placeholder
automatically — no broken-image icon, no error in the console.
